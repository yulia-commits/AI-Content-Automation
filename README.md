# AI Content Automation Tools
Коллекция инструментов на базе Python и Gemini API для автоматизации работы с текстом.

## В этом репозитории:
1. **Генератор тестовых статей (01_article_test_generator)** — создание текстов по заданным параметрам.
2. **Анализатор отзывов (02_reviews_analyzer)** — автоматическая обработка отзывов и экспорт в Google Таблицы.

# Анализатор отзывов через Gemini API

Скрипт для автоматической обработки отзывов из текстового файла и их выгрузки в Google Таблицы.

## Что делает проект:
- Читает отзывы из файла reviews.txt.
- Отправляет их в нейросеть Gemini для определения тональности и категории.
- Сохраняет результат анализа в отдельный файл.
- Записывает готовые данные в Google Таблицу.

## Инструменты:
- Python
- Google Gemini API (модель 2.0 Flash)
- Библиотека gspread для работы с таблицами

## Как запустить:
Код предназначен для запуска в Google Colab. Нужно загрузить файл с отзывами, вставить свой API ключ и ссылку на таблицу.
