# AI Content Automation

Набор Python-инструментов для автоматизации обработки текстов с использованием Google Gemini API.

## Tech Stack

- Python
- Google Gemini API
- Google Colab
- Google Sheets
- gspread
- Prompt Engineering

## Проекты

### Анализатор отзывов

Автоматический анализ пользовательских отзывов с помощью Gemini API.

Возможности:

- анализ тональности;
- определение категории;
- экспорт результатов в Google Sheets.

---

### Генератор статей

Генерация текстов по заданным параметрам с использованием Gemini API.

Возможности:

- генерация статей;
- настройка промптов;
- Few-shot Prompting.

---

## Результат

Проект демонстрирует использование Gemini API для автоматизации обработки текстовой информации и интеграции результатов с Google Sheets.
